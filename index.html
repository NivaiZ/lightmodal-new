LightModal — Showcase

Всё в одном: dialog fallback, inline, media, gallery, drag-to-close, idle UI, a11y, AJAX/JSON, события и API.

Inline (перенос DOM внутрь модалки)

Контент живой: формы, фокус, скролл внутри.

Подсказка: попробуй Tab/Shift+Tab, ESC, клик по backdrop, а на touch — drag-to-close.

API (JS)

Открытие по кнопке, опции, closeExisting и события.

События: lightmodal:open / lightmodal:close (с detail.instance).

Галерея

Группировка по data-gallery, подписи, счётчик, навигация, swipe.

Клавиатура: ← →, Escape. Опции: data-lm-loop, data-lm-idle.

Видео / iframe

YouTube/Vimeo/Rutube/VK распознаются по URL, иначе — iframe.

YouTube auto Vimeo auto Iframe type=iframe
Для iframe лучше задавать ширину/высоту через data-lm-width/data-lm-height.

AJAX (HTML)

Загрузка HTML, вырезание фрагмента по селектору.

Открыть HTML type=ajax Только фрагмент ajaxSelector
Есть базовая санация ответа (не замена DOMPurify для недоверенных источников).

JSON + ajaxSuccess

Рендерим JSON в красивую карточку (или покажем <pre> по умолчанию).

JSON по умолчанию <pre>

Темы / анимации

Тема задаётся на контейнере через data-theme.

Dark + zoom-in Light + slide-up Auto + fade
Можно менять внешний вид через CSS-переменные --lm-* (см. readme.md).

Простой inline контент

Этот контент переносится в модалку как DOM-узел (не строка).

Проверь: ESC / клик по фону / Tab navigation / возврат фокуса после закрытия.

Форма обратной связи

Сообщение отправлено

Это демонстрация: по submit мы закрываем текущую модалку и открываем следующую.

Длинный прокручиваемый контент

Скролл должен быть внутри модалки, а фон — заблокирован.

Что есть в LightModal

  • <dialog> где доступен + fallback на <div>
  • Scroll lock с авто-детектом Lenis
  • Focus trap, restoreFocus
  • Drag-to-close, idle-режим
  • AJAX/JSON, базовая санация
  • DOM события: lightmodal:open / lightmodal:close
